OrangePi Control Camera Python OpenCV

1 Prepare the System

1.1 SSH setup

mkdir -p ~/.ssh
cat >> ~/.ssh/authorized_keys

Paste your open SSH key and press Ctrl+D.

1.2 Passwordless sudo

echo 'orangepi ALL=(ALL) NOPASSWD: ALL' | sudo tee -a /etc/sudoers.d/010_orangepi-nopasswd

1.3 Time

sudo timedatectl set-timezone America/New_York
sudo perl -n -i.orig -e 'next if /^pool [1-3]/;s/0\.[a-z.]+\.ntp\.org|ntp\.ubuntu\.com/softcomputer.com/;print' /etc/ntp.conf
sudo systemctl restart ntp

1.4 Update packages

sudo apt update
sudo apt upgrade

2 Build Kernel 3.4.113

cd ~ && git clone --depth=1 --branch=master https://github.com/orangepi-h3-linus/orangepi_h3_linus.git

cd ~/orangepi_h3_linus/OrangePi-Kernel/linux-3.4.113
make -j$(nproc) ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- uImage modules
sudo make ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- modules_install
sudo mv /media/boot/uImage /media/boot/uImage.3.4.112
sudo cp arch/arm/boot/uImage /media/boot

sudo reboot

3 Build v4l2loopback

cd ~ && git clone --depth=1 --branch=master https://github.com/umlaeute/v4l2loopback
cd ~/v4l2loopback/
make
sudo make install
sudo depmod -a

4 Load Modules

sudo modprobe gc2035 hres=0
sudo modprobe vfe_v4l2
sleep 5
sudo modprobe v4l2loopback

To have them auto loaded upon reboot: 

echo -e "gc2035\nvfe_v4l2\nv4l2loopback" | sudo tee -a /etc/modules

5 Build and Run vidcopy

sudo apt install libv4l-dev
cd ~ && git clone https://github.com/lhelontra/vidcopy
cd ~/vidcopy/
gcc vidcopy.c -o vidcopy
./vidcopy -w 800 -h 600 -r 30 -i /dev/video0 -o /dev/video1 -f UYVY &
